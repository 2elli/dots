# sway
## exit
`sway exit`

## list outputs, inputs
`swaymsg -t get_<outputs|inputs>`


# daemons
## notifs
`swaync`

## auto tiling
`autotiling-rs`


# utils
## launcher
`tofi-drun`

## configure displays
`nwg-displays`

## screenshots
`grimshot`

## lock
`swaylock`

## clipboard
`wl-copy` and `wl-paste`
aliased to `wlc` and `wlp`
